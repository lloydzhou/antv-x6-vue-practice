提供一个vue下使用X6的最佳范本：
1. 将函数式的X6Graph组件化
2. 简单易用，易于做逻辑拆分  
3. 支持多实例  
4. 方便在自己的组件中管理图数据  
5. 自动检测图数据变化，并增量更新到x6画布  
6. 内部使用batchUpdate优化性能

## 安装
```
npm install vue-x6-graph
yarn add vue-x6-graph
```
